Największy projekt

SMMU — System Monitorowania Maszyn i Urządzeń
FAKRO · SCADA / monitoring · wdrożenie etapowe (w trakcie)

System do akwizycji danych w czasie rzeczywistym z maszyn i czujników, wizualizacji w aplikacji webowej oraz budowy bazy danych do analiz trendów i utrzymania ruchu.

Backend: Java (mikroserwisy) + REST API Frontend: Angular (web) InfluxDB (time-series) MS SQL (konfiguracje, użytkownicy) OPC UA · MQTT · ADS Git · SQL
  • Zbieranie i normalizacja danych z urządzeń (PLC Beckhoff / Fanum oraz czujniki przemysłowe) z użyciem OPC UA, MQTT i ADS.
  • Wizualizacja: synoptyki, trendy, tabele; dostęp z poziomu przeglądarki.
  • Archiwizacja: dane dynamiczne w InfluxDB, dane statyczne/konfiguracyjne w MS SQL.
  • Administracja: dodawanie/edycja urządzeń, zarządzanie połączeniami, role i uprawnienia operatorów.
  • Plan rozwoju: alarmy z progami i priorytetami, raportowanie (PDF/Excel), integracja MES/ERP, opcjonalnie kolejki (RabbitMQ).
(Opis na bazie Twojej dokumentacji funkcji i dokumentacji rozszerzonej systemu SMMU.)
